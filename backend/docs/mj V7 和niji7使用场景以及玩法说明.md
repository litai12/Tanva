Midjourney V7 核心能力
1. Omni-Reference 全能参考系统
多元素锁定：支持人物、物体、车辆、动物等多种主题元素参考，不仅限于人脸 
权重精细控制：通过 --ow 参数（0-1000）调节参考强度：
25-50：风格迁移，保留氛围但弱化具体特征
400-1000：高度还原，适用于影视海报、虚拟角色设计 
操作方式：Web端直接拖拽至 omni-reference 区域，或命令行使用 --oref [URL] --ow [权重]
2. 精准文字渲染
英文文字识别准确率从V6的30%提升至75% 
使用引号包裹文字即可生成，如 "SOLSTICE COFFEE" 
手部畸形改善率达85% 
3. 草稿模式（Draft Mode）
成本减半，渲染速度提升10倍 
对话式交互：点击草稿模式后，提示栏变为对话模式，可直接说"将猫换成猫头鹰"或"改为夜晚场景"，AI自动调整提示并重绘 
语音模式：开启麦克风后可语音实时创作，"让图像像流动的梦境一样在手下生成" 
适用于快速迭代创意，确认后再点击"增强"以完整质量重绘
4. 模型个性化（Personalization）
V7首个默认启用个性化的版本，需约5分钟解锁训练 
学习用户审美偏好，可随时开关
5. 风格参考增强（--sref）
支持传入多张图片URL模仿整体美学风格
Style Tuner生成独特风格代码，确保系列作品风格统一 
6. 角色一致性（--cref）
通过角色图片URL保持不同场景中的外貌特征一致
适用于虚拟偶像、系列插画、故事板创作 
🌸 Niji 7 核心能力（动漫专项）
1. 动漫细节极致优化
水晶般清晰的眼部渲染：瞳孔、反光、高光对齐精准，告别模糊虹膜 
更扁平、更干净的线条艺术：强调线稿感，接近真实动画制作风格 
背景元素精度提升：如飘落的花瓣与主角同等精细度
2. 提示词忠实度大幅提升
从V6的"氛围理解"转向字面理解 
复杂描述如"绿发蓝眼四臂女孩手持冰淇淋"能精确还原
特定姿势、多配件、非标准特征均可控
3. 风格关键词体系
"anime screenshot"：生成真实动画截图感画面 
"minimalist graphic logo" / "flat shading"：极简平面风格
"90s anime" / "cel-shaded"：赛璐璐复古风格
4. 与V7风格互通
支持使用 --sref 调用Midjourney V7生成的风格代码，在动漫语境中重新诠释 
5. 暂不支持但即将推出的功能
角色参考（--cref）：团队正在开发"更特别的替代方案" 
个性化与情绪板：将在后续更新中推出

💡 进阶使用技巧
V7 提示词工程
plain
复制
[主体] + [环境] + [光照/氛围] + [视角/构图] + [艺术风格] + [参数]

示例：
"穿西装的亚洲男性握手，背景现代办公室，体积光照明，低角度特写，电影级写实风格，文字'2025 TECH'清晰显示在背景屏幕上 --ar 16:9 --v 7 --cref [参考图URL] --ow 600"
Niji 7 提示词结构
plain
复制
[具体角色描述] + [场景动作] + [风格关键词] + [构图]

示例：
"短发绿发蓝眼少女，四只手臂各持不同口味冰淇淋，夏日祭典背景，anime screenshot风格，正面中景，cel-shaded --niji 7 --sref [风格代码]"
跨模型工作流
概念探索：用V7 Draft Mode快速生成写实概念草图
风格转换：将V7生成的风格代码（--sref）应用到Niji 7，获得"写实构图+动漫风格"的融合效果 
角色统一：V7的--cref生成角色参考图，Niji 7通过详细文字描述+风格参考实现动漫化（暂需手动描述，等待Niji 7 cref功能上线）
国内平台优化方案
通过夸克"造点"等平台使用V7时：
利用滑块控制替代手动输入权重（风格化/怪异化/多元化）
使用智能润色和特征词库（120+视角/光线/构图预设）降低提示词门槛 
夸克图片1.0针对亚洲人像、中文文字、国风元素优化，适合电商和本土营销
