悠船API当前支持的模型版本介绍。

悠船生图API为您提供多样化的模型版本选择，涵盖v6系列、v7系列以及niji系列。每个模型版本在图像生成质量、功能特性和参数支持方面都具有独特优势。本指南将深入解析各版本的核心特点、支持的功能参数以及版本间的关键差异，助您根据具体应用场景选择最优模型方案，充分发挥API的潜力。

-   **模型版本**：API目前支持模型 `v6`、`v6.1`、 `v7`、`niji 6`、`niji 7`
    
-   **默认模型**：`v6.1`
    

## **niji7**

niji V7 是专为动漫创作打造的重大版本升级，凭借更精准的角色控制、更丰富的风格化选项以及更出色的图像质量，为动漫创作者和爱好者提供专业级 AI 艺术创作体验。

核心亮点：

-   **全新提示系统**：引入“敏感模式”，生成细节更丰富、光影更具戏剧张力的图像。
    
-   **精细风格控制**：支持在“原始写实”与“高度风格化”之间灵活调节，满足多样化创作需求。
    
-   **深度文本理解**：显著提升对复杂角色描述的理解与精准渲染能力。
    
-   **多角色一致性**：大幅改进多角色场景中的形象稳定性，确保角色特征高度统一。
    
-   **全面美学升级**：在图像质量、动态表现力、构图平衡及色彩层次方面均实现质的提升。
    

## **版本7**

版本7（2025年4月3日发布）在图像生成领域实现了重大突破。该版本显著提升了文本和图像提示的处理精度，同时在图像质量上实现了质的飞跃——特别是在人体结构、手部细节和物体纹理方面，呈现出前所未有的丰富度和连贯性。

作为核心创新，V7引入了两大革命性功能：

-   [草图模式](https://help.aliyun.com/zh/marketplace/youchuan-api-speed-mode#e782da7e37fsm)支持快速迭代生成
    
-   [万物引用](https://help.aliyun.com/zh/marketplace/youchuan-api-advanced-feature-image-reference#万物引用)提供了更强大的图像参考能力，为创意工作流带来全新可能

## **版本6.1**

版本6.1于2024年7月30日发布，作为当前的默认模型。它生成的图像具有更连贯的效果，细节和纹理更精确，并且生成图像的速度比版本6快约25%。

## **版本6**

版本6于2023年12月20日发布，并从2024年2月14日至7月30日作为默认模型。版本6增强了对较长输入的提示准确性，改进了连贯性和知识，并提供了先进的图像提示和混音功能。

## **Niji 6**

Niji模型是悠船内的一个特殊系列，与Spellbrush合作开发，专注于东方和动漫美学以及插画风格。

Niji 6模型于2024年6月7日发布，具有改进的日文文本渲染功能，能有效处理短日语假名文本，甚至能渲染简单的汉字。它提供了更好的图像细节，特别是在动漫眼睛结构等元素方面，并解决了影响先前生成的一小部分图像的某些轻微图像瑕疵问题。

### **参数验证规则**

| **参数类型** | **验证规则** |
| --- | --- |
| 风格引用(sref) | 最多使用20个sref；对于niji6/v6.\\*模型，仅在sv=4的时候支持random值 |
| 角色引用(cref) | 不支持指定单张图片权重，仅支持图片URL |
| 草稿模式(draft) | 不支持与tile/oref同时使用 |
| 输出引用(oref) | 不支持指定多张图片，必须为图片URL |
| V7模型 | 不支持多提示词 |
| 图片提示词 | 最多使用20张垫图 |
| 文本提示词 | 最多使用7个多提示词 |
| 怪异参数(weird) | 不支持与personalize参数一起使用 |
| 原始模式(raw) | 在执行视频任务时，需要搭配文本提示词使用 |

### **参数依赖关系**

| **参数** | **依赖条件** |
| --- | --- |
| sv (风格版本) | 需要搭配sref使用 |
| sw (风格权重) | 需要搭配sref使用 |
| cw (角色权重) | 需要搭配cref使用 |
| iw (图片权重) | 需要搭配垫图使用 |
| ow (输出权重) | 需要搭配oref使用 |
| sref (风格引用) | 需要搭配文本提示词使用 |
