# Tanva - 专业绘图与AI创作平台

基于React和TypeScript的现代化专业绘图应用程序，集成AI图像生成功能。

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 主要功能

- 🎨 专业绘图工具（基于Paper.js）
- 🤖 AI图像生成（Gemini 2.5 Flash）
- 🖼️ 图像编辑和融合
- 📱 现代化响应式界面
- 🎛️ 双模式操作（聊天模式 + 节点模式）

## 技术栈

- **前端**: React 19 + TypeScript + Vite
- **UI**: Tailwind CSS + shadcn/ui
- **绘图**: Paper.js + Canvas API
- **状态管理**: Zustand
- **AI集成**: Google Gemini API
