# Director Console / LibTV 1:1 功能矩阵

> 唯一完成标准：Tanva Director Console 节点从画布外壳、打开体验、全部内部功能到输入/输出与保存重载，均有 LibTV 在线证据和本地浏览器回归证据。仅有按钮、仅有字段或仅通过构建不算完成。

## 必须删除的 Tanva 遗留

- 用户可见入口已移除：灰模 MP4、镜头片段、焦距/景深、飞行录制、环绕、动作库、群演广播等。
- `DirectorConsoleModal` 已物理删除灰模 MP4 编码、视频上传/节点输出、15 秒拆分、旧 shot timeline、环绕预览、飞行录制与群演属性广播执行链。
- `DirectorCaptureRunner` 已删除隐藏的 Agent `clip` 灰模视频执行分支；`sendToCanvas` 现在只允许输出 LibTV 摄像机截图图片。
- 已删除旧 `CameraPropertiesPanel`、导演台视频上传器、飞行录制和旧时间线逐帧渲染文件。
- `Viewport` 与旧兼容 state 中仍有未被生产入口调用的动画逐帧/路径辅助代码，完成前继续收口或确认属于 LibTV 轨迹所需实现。
