# 技术设计: 上传中图片仅允许移动

## 技术方案
### 核心技术
- React + TypeScript
- Paper.js 画布交互

### 实现要点
- 使用 `imageData.pendingUpload` 作为“上传中”判定标准。
- 在交互层（`useInteractionController`）放行拖拽移动，但拦截 resize/rotate/delete/编辑等操作。
- 在 UI 层（`ImageContainer` 工具栏）对上传中图片禁用按钮，避免触发编辑流程。
- 在键盘快捷键与上下文行为（`DrawingController`）中屏蔽上传中图片的删除与变形操作。

## 架构设计
无架构变更。

## API设计
无 API 变更。

## 数据模型
无数据变更。

## 安全与性能
- **安全:** 仅调整前端交互，不新增外部输入或存储。
- **性能:** 仅增加少量条件判断，不引入额外渲染负担。

## 测试与部署
- **测试:** 手动验证上传中图片可拖拽，其他操作被禁用；上传完成后恢复全部操作。
- **部署:** 随前端发布。
