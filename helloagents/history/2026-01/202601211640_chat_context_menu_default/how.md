# 技术设计: 对话框右键菜单恢复为浏览器默认

## 技术方案
### 核心技术
- React 19 + TypeScript

### 实现要点
- 移除对话框内容区域的 `onContextMenu` 拦截逻辑，避免 `preventDefault`/`stopPropagation`。
- 移除 `ContextMenu` 状态与渲染节点，确保不再生成自定义菜单。

## 安全与性能
- **安全:** 不再处理剪贴板导入导出，降低误触风险
- **性能:** 减少事件处理与组件渲染开销

## 测试与部署
- **测试:** 手动验证对话框区域右键行为
- **部署:** 前端正常构建发布即可
