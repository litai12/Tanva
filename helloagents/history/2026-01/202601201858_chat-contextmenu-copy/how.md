# 技术设计: AI 对话框右键复制选中文本

## 技术方案
### 核心技术
- React + TypeScript
- DOM Selection API (`window.getSelection`)

### 实现要点
- 在 `AIChatDialog` 的 `handleChatContextMenu` 中读取 `window.getSelection()`。
- 若存在非空选中文本且选区位于对话框内容区，则直接返回，允许浏览器默认右键菜单。
- 否则保持现有逻辑：阻止默认菜单并展示自定义菜单。

## 安全与性能
- **安全:** 不引入新数据流或权限边界。
- **性能:** 右键事件中仅做一次选区检查，开销可忽略。

## 测试与部署
- **测试:** 手动验证对话框文本选中右键菜单与无选中文本时自定义菜单。
- **部署:** 前端常规发布流程。
